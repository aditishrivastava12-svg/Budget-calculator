Budget Calculator

About

  1. Objective:

    2. The objective of your project is to create a budget calculator that helps users track their income and expenses, providing them with a clear overview of their financial situation.

  2. Technologies Used:

    3. HTML: For creating the structure and layout of the web page.

    CSS: For styling and formatting the visual appearance of the elements.

    JavaScript: For adding interactivity and functionality to the application.

  3. Features:

    4. User Authentication:Users can log in to access the budget calculator functionality.

    Add Income: Users can add their sources of income, providing a title and the income amount.

    Add Expense: Users can add their expenses, providing a title and the expense amount.

    Display Balance:The application calculates the balance by subtracting the total expenses from the total income and displays it.

    Income and Expense Lists:The application shows separate lists for income and expense entries.

    Switch Between Income and Expense Views:Users can switch between views to see their income or expense entries only, or view both together.

    Delete Entries:Users can delete any income or expense entry they've added.

    Edit Entries: Users can edit existing income or expense entries to modify their titles or amounts.

    Data Persistence:You have implemented the functionality to save the income and expense data in the browser's local storage so that it persists even when the page is refreshed or closed.
